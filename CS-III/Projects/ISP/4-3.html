Technical Challenges


There were some complications with the new way to link css files on our project with the new login/signup systems. There's a new syntax that we have to use because we are using flask. In order to solve this issue, we had to use an entire different folder named "static" and put all of the functions and css files (ex: index.js or dashboard.css) and we had to use a url_for command inside of our html files to connect the css to the main files. Another issue that is going on is when William and I created our flask servers, we did them 2 different ways, so we are working to integrate all of our work into the same area so that they both work and our project can keep on progressing.
